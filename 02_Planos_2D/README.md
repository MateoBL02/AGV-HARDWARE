# 02 — Planos de Fabricación 2D

Planos técnicos con cotas, tolerancias y materiales para la fabricación de componentes del AGV.

## Requisitos de cada plano

- Formato: **PDF**
- Escala indicada
- Cotas y tolerancias dimensionales completas
- Material especificado
- **Rótulo institucional** — Universidad de Pamplona
  - Nombre del proyecto: AS/RS Automatizado
  - Grupo: G2 — AGV Hardware
  - Nombre del componente
  - Fecha, revisión y dibujante

## Contenido esperado

| Archivo | Componente |
|---------|-----------|
| `plano_chasis.pdf` | Chasis principal del AGV |
| `plano_plataforma_elevacion.pdf` | Plataforma de elevación de carga |
| `plano_soporte_motores.pdf` | Soporte de motores de tracción |
| `plano_soporte_sensores.pdf` | Soporte del arreglo de sensores QTR-8A |
| `plano_ensamble_general.pdf` | Vista de ensamble general con referencias |

> Agregar planos adicionales según las piezas finales del diseño.
